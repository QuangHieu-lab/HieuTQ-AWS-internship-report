### Week 11 Objectives:

* Gain hands-on experience deploying and managing Red Hat OpenShift Service on AWS (ROSA).
* Learn how to create, configure, and manage an OpenShift cluster on AWS.
* Practice deploying applications into an OpenShift-managed Kubernetes environment.
* Learn how to set up a basic CI/CD workflow for automated deployments.
* Practice cleaning up ROSA clusters and related AWS resources efficiently.
* Gain deeper understanding of Infrastructure as Code (IaC) using AWS EKS Blueprints and CDK.
* Learn how to create EKS clusters using Blueprints and manage infrastructure as reusable constructs.
* Practice building deployment pipelines and integrating ArgoCD for GitOps workflows.
* Explore how to manage teams, roles, and add-ons using IaC.
* Perform cleanup for CDK- and EKS-created resources.
* Explore S3 storage types and understand use cases for each class.
* Practice working with S3 lifecycle rules for data transition and archiving.
* Learn more about CloudFront behaviors, caching, and distribution configuration.
