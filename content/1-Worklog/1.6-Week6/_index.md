### Week 6 Objectives:

* Strengthen understanding of AWS Identity and Access Management (IAM) by creating restriction policies and limited user accounts.
* Learn about AWS global infrastructure, including Regions, Availability Zones (AZs), and Edge Locations.
* Explore AWS storage services such as EC2 Instance Store, Amazon Elastic Block Store (EBS), Amazon Elastic File System (EFS), and Amazon FSx.
* Study AWS database services, including Amazon RDS, Amazon Aurora, Amazon DynamoDB, and Amazon ElastiCache.
* Gain knowledge of AWS Artificial Intelligence (AI), Machine Learning (ML), and Data Analytics services, including Amazon Comprehend, Polly, Transcribe, Translate, Rekognition, Textract, Lex, Kendra, Personalize, Bedrock, Q Business, and Q Developer.
* Continue developing hands-on experience using both AWS Management Console and AWS Command Line Interface (CLI) to manage and configure resources.
