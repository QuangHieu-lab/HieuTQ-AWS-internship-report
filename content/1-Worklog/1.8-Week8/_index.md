### Week 8 Objectives:

* Gain a solid understanding of AWS fundamentals, including the main service categories: Compute, Storage, Networking, and Databases.
* Learn how to navigate and utilize the AWS Management Console effectively.
* Practice managing IAM users, roles, groups, and policies to control access and permissions
* Learn how to use AWS CLI for performing resource management operations
* Explore and practice working with S3, EC2, RDS, VPC, and CloudTrail
* Understand security and encryption through Key Management Service (KMS)
* Gain experience with Athena to query and analyze data stored in S3
* Learn how to deploy applications, perform backups, and clean up AWS resources efficiently
