### Week 1 Objectives:

* Get introduced to the First Cloud Journey program and connect with team members.  
* Understand the fundamentals of AWS Cloud and how to navigate the AWS Management Console and CLI.  
* Learn and practice Identity and Access Management (IAM) for user, group, and role configuration.  
* Explore Elastic Compute Cloud (EC2) and understand how to launch and manage virtual instances.  
* Gain hands-on experience with budget management in AWS using cost and usage budgets.  
* Learn about AWS Support Plans and how to manage support requests.  
* Understand and configure basic networking concepts in AWS, including VPC, Subnets, Internet Gateway, Route Tables, Security Groups, and NAT Gateway.

### Week 1 Achievements:

* Successfully created and configured an AWS Free Tier account.  
* Became familiar with the AWS Management Console, including service navigation and resource management.  
* Practiced creating and managing IAM users, groups, and roles, and tested role switching.  
* Launched and configured EC2 instances, established secure connections, and tested accessibility.  
* Set up budget templates, including cost, usage, and savings plan budgets to monitor AWS spending.  
* Explored AWS Support Plans, created and managed support cases, and understood support tiers.  
* Designed and deployed a Virtual Private Cloud (VPC) with:  
  * Subnets (public/private)  
  * Internet Gateway and Route Table for outbound routing  
  * Security Groups and Network ACLs for secure communication  
  * NAT Gateway and EC2 Instance Connect Endpoint for connectivity management  
* Gained a solid understanding of how networking and security integrate with EC2 instances and other AWS services.
