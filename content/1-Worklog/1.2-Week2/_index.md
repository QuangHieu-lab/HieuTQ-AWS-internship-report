### Week 2 Objectives:

* Learn and configure Hybrid DNS using Amazon Route 53.
* Practice setting up Hybrid DNS with Route 53, including creating inbound and outbound endpoints and testing name resolution.
* Practice creating VPC peering connections and enabling cross-VPC DNS resolution
* Learn about and set up an AWS Transit Gateway to simplify and centralize VPC connectivity
* Understand and deploy Amazon EC2 concepts, including AMIs, EBS, and backups.
* Practice implementing AWS Backup to automate backup and restore processes for EC2 instances.
