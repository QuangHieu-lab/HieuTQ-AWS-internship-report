### Week 10 Objectives:

* Understand the end-to-end workflow of building a **Data Lake** on AWS, including data preparation, ingestion, querying, and visualization.
* Learn how to use **AWS Glue**, **Glue Studio**, and **Glue DataBrew** for ETL, data cataloging, and data transformation.
* Gain hands-on experience with **Amazon Athena** for querying structured and semi-structured data stored in S3.
* Develop skills in creating dashboards, interactive visuals, and insights using **Amazon QuickSight**.
* Practice performing advanced analytics using **Kinesis Data Analytics** and integrating analytical insights across AWS services.
* Learn how to deploy and serve data-driven applications using **AWS Lambda**, Redshift, and related services.
* Understand and apply cloud security through **Security Groups**, **AWS Firewall Manager**, and proper security auditing.
* Gain exposure to AWS AI/ML services and learn the fundamentals of Generative AI with **Amazon Bedrock**.
* Follow AWS best practices for resource cleanup, cost management, and environment hygiene after each lab.
