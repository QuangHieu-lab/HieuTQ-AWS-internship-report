### Week 7 Objectives:

* Learn and explore **AWS Data Analytics Services**, including Data Pipeline, Kinesis, Firehose, Redshift, Glue Data Catalog, QuickSight, and OpenSearch.  
* Understand key **AWS Security and Protection Services**, such as Secrets Manager, Systems Manager, Shield, WAF, KMS, Macie, ACM, Inspector, GuardDuty, and Security Hub.  
* Explore **Monitoring, Compliance, and Governance Services** including CloudWatch, CloudTrail, Artifact, AWS Organizations, Control Tower, Service Catalog, and Trusted Advisor.  
* Learn about **AWS Pricing and Support**, focusing on pricing concepts, billing services, and cost management tools (Budgets, Cost Explorer, and Pricing Calculator).  
* Strengthen skills in using the **AWS Management Console** and **AWS CLI** for managing and monitoring AWS resources.

### Week 7 Achievements:

* Successfully connected with peers in the **First Cloud Journey** program and exchanged learning experiences.  
* Gained a comprehensive understanding of core **AWS Service Groups**, including Compute, Storage, Networking, and Database.  
* Explored and practiced using **AWS Data Analytics tools** (Kinesis, Redshift, Glue, and QuickSight) for data processing and visualization.  
* Learned to secure AWS environments using **Secrets Manager, Shield, WAF, KMS, and GuardDuty** to protect data and applications.  
* Practiced using **CloudWatch** for creating metrics, alarms, dashboards, and logs to monitor system performance.  
* Studied **CloudTrail** for auditing service activities and **AWS Artifact** for managing compliance reports and agreements.  
* Understood **AWS Governance tools** such as Organizations, Control Tower, and Service Catalog to manage multiple accounts effectively.  
* Explored AWS **Pricing and Billing tools** (Budgets, Cost Explorer, and Pricing Calculator) to optimize and estimate cloud spending.  
* Installed and configured **AWS CLI**, including setting up Access Key, Secret Key, and Default Region.  
* Performed basic operations via CLI such as checking configurations, retrieving region lists, viewing EC2 information, and managing key pairs.  
* Developed the ability to use both **Console and CLI** in parallel for managing AWS services efficiently.
