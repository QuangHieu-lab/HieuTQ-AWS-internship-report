### Week 4 Objectives:

* Learn about AWS fundamental services and how to interact with them using both the Management Console** and AWS CLI.  
* Understand how to create, manage, and back up cloud infrastructure using AWS.  
* Gain experience in setting up virtual environments with VMware and Ubuntu for testing.  
* Explore AWS Security Services such as IAM, Cognito, and AWS Organizations.  
* Practice file system management (EFS) with performance testing and scaling.

### Week 4 Achievements:

* Deployed and tested S3 storage operations including backup and restoration.  
* Installed VMware Workstation and Ubuntu, and practiced VM export, import, and AMI deployment.  
* Gained a solid understanding of AWS Security Services (IAM, Cognito, and Organizations).  
* Set up and managed EFS Multi-AZ file systems, tested SSD vs HDD performance.  
* Applied monitoring and optimization techniques: data deduplication, shadow copies, quotas, and scaling.  
* Practiced using AWS CLI for account verification, region listing, EC2 management, and key pair configuration.  
* Strengthened the ability to manage and automate AWS resources through both the Console and CLI.
