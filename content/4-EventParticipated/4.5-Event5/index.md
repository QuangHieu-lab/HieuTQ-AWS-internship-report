# Summary Report: ​AWS Edge Services Workshop
### Event Objectives
- Understand how to architect and optimize global content delivery using Amazon CloudFront
- Learn core CDN capabilities, including edge locations, caching strategies, and performance optimization
- Explore security controls and best practices for content delivery at scale
- Understand AWS WAF and application protection to secure web applications

- **Nguyen Gia Hung** – Head of solution architect
- **Julian ju** – Senior Edge Services specialist solution architect
### Coordinators
- **Kevin Lim** – Event supporter
## Key Highlights

### From Edge to Origin: CloudFront as Your Foundation
- Learned how to design and optimize global content delivery using Amazon CloudFront.
- Explored CDN capabilities, edge locations, and integrating security controls at scale.
- Session targeted towards cloud architects and developers building performant global architectures.

### Attack Surface Defense: WAF & Application Protection
- Gained insights into securing web applications with AWS WAF. 
- Learned how to block malicious traffic, enforce application-layer protection, and mitigate OWASP Top 10 vulnerabilities.
- Covered strategies to defend against bot attacks and enhance application security posture.
### Hands-On Workshop: Optimize Internet Web Application
- Practical lab session guided by AWS specialists.
- Strengthened understanding through implementation-focused exercises.
