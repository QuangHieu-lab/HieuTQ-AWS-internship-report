# Summary Report: ​AI/ML/GenAI on AWS (AWS Cloud Mastery Series #1)
### Event Objectives

- Introduce participants to key AWS AI/ML services including SageMaker, Rekognition, Textract, Bedrock, and more  
- Showcase real-world applications of Computer Vision, NLP, and Generative AI  
- Provide practical knowledge through demos and hands-on sessions  
- Highlight how AWS AI services can accelerate modern development workflows and improve efficiency

- **Lam Tuan Kiet** – Sr DevOps Engineer FPT SoftWare
- **Danh Hoang Hieu Nghi** – AI Engineer Renova Cloud
- **Dinh Le Hoang Anh**– Cloud Engineer Trainee First Cloud AI Journey

## Key Highlights

### Amazon Rekognition  
- Image & video analysis, object detection, unsafe content moderation  
- Facial analysis, comparison, and emotion detection  
- Ideal for security, media analysis, and automation workflows  

### Amazon Textract  
- Intelligent document OCR  
- Extracts text, tables, forms, and handwriting  
- Used for invoice automation, onboarding forms, and document processing  

### Amazon Lookout for Vision  
- Detects visual defects using machine learning  
- Useful for manufacturing, quality control, and anomaly detection  

### Amazon Translate  
- Real-time machine translation  
- Multi-language support with automatic detection  
- Perfect for multilingual apps, content localization, and chat systems  

### Amazon Polly  
- Converts text to natural speech  
- Large library of voices and languages  

### Amazon Kendra  
- Intelligent enterprise search engine  
- Semantic search using NLP for accurate answers  
- Integrates with S3 and internal systems  

### Amazon Personalize  
- Real-time personalized recommendations  
- Used for product, content, and ranking predictions  

---

## The Key Takeaways

- AWS provides **powerful pre-built AI/ML services** requiring little ML knowledge  
- These services help automate vision, text processing, translation, and personalization  
- They integrate easily with S3, Lambda, Bedrock, and enterprise systems  
- Combining these services enables **end-to-end intelligent applications**  
- Scalable and ready for both startups and large enterprises
