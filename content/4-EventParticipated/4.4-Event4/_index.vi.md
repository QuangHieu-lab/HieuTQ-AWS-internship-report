## Nội dung nổi bật

### AWS CodeCommit
- Dịch vụ kho Git an toàn và có khả năng mở rộng trên AWS  
- Hỗ trợ GitFlow, Trunk-Based, và feature-branch workflows  
- Lý tưởng cho hợp tác, quản lý phiên bản, và tự động hóa quy trình  

### AWS CodeBuild
- Dịch vụ build được quản lý đầy đủ để biên dịch, kiểm thử, và đóng gói code  
- Hỗ trợ cấu hình buildspec tùy chỉnh và build song song  
- Tạo điều kiện cho pipeline kiểm thử tự động trong CI/CD  

### AWS CodeDeploy
- Tự động triển khai ứng dụng trên EC2, Lambda và on-premises  
- Hỗ trợ chiến lược triển khai Blue/Green, Rolling, và Canary  
- Giảm downtime và rủi ro triển khai nhờ các cơ chế điều phối traffic  

### AWS CodePipeline
- Tự động hóa CI/CD end-to-end, tích hợp với các công cụ DevOps trên AWS  
- Tự động hóa các giai đoạn build, test và deploy với quản lý phiên bản đầy đủ  
- Lý tưởng cho continuous delivery, pipeline đa môi trường và tự động hóa workflow  

### AWS CloudFormation
- Templates khai báo để provision và quản lý tài nguyên AWS  
- Quản lý stack với khả năng phát hiện drift và rollback  
- Đảm bảo hạ tầng tự động hóa nhất quán và có thể tái tạo  

### AWS CDK (Cloud Development Kit)
- Infrastructure as Code sử dụng các ngôn ngữ lập trình quen thuộc  
- Các construct cao cấp và pattern tái sử dụng cho phát triển nhanh  
- Tích hợp liền mạch với CloudFormation để triển khai  

### Dịch vụ Container trên AWS
- Kiến thức cơ bản về Docker cho microservices và containerization  
- Amazon ECR để lưu trữ, quét và quản lý lifecycle của image  
- Amazon ECS & EKS để triển khai container có khả năng mở rộng và orchestration  
- AWS App Runner để host ứng dụng container một cách đơn giản  

### Giám sát & Khả năng quan sát
- Amazon CloudWatch: metrics, logs, alarms, và dashboards  
- AWS X-Ray: phân tích phân tán và tracing hiệu năng  
- Cho phép giám sát full-stack và quan sát hoạt động  

### Best Practices DevOps & Case Studies
- Chiến lược triển khai: feature flags, A/B testing, và canary releases  
- Tích hợp kiểm thử tự động trong pipeline CI/CD  
- Quy trình quản lý sự cố và postmortem hiệu quả  
- Case study: chuyển đổi DevOps cho startup và doanh nghiệp
